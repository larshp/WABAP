Proof-of-concept.

### Building/Installing

* `npm install`

* `npm test`

* `npm run zip`

* download build.zip

* clone abap using [abapGit](http://www.abapgit.org)

* run ZWABAP_INSTALLER and supply with build.zip (install into new isolated package)

* run via SICF service ZWABAP

* endpoint is located at zwabap/rest/
